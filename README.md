# Accounting-Ledger-Application
This is a CLI application. With this application, you can track all financial transactions for your business or for your own personal use. It is a three-layered application that starts on the home screen with options for deposits, payments, and displaying the ledger. Then comes the Ledger Screen and all its functionalities. Finally, the Report Screen.

Welcome to Accounting-Ledger-Application!!!

The goal of this application is to keep track of your finances. It could be used for your personal finances or for your buisiness accounting purposes. It has three layers: Home-Screen, Ledger-Screen, and finally the report screen.

When the application is ran, you will see a menu like such:

Welcome to Accounting-Ledger-Application!! 
Type (D) to Add Deposit
Type (P) to Make Payment
Type (L) to go to Ledger Screen
Type (X) to exit

Please Enter one of the above options: 

    1. It asks you to enter one of the following options. If you type D, you will be prompted to enter information for     your deposit: 
    
      Please Enter one of the above options: d
      Please enter the Description of your deposit: Computer software update
      Please enter the Vendor name: Dell
      Please enter the amount here: 100
    
      once you have entered your information, this will be added to the transactions.csv.
    
     
    
    2. If you type "p" you will be prompted to a similar screen that asks you questions on a payment that you will be entering in your transaction.csv file. 
    
    3. "L" takes you to the ledger page.
            Once you enter the ledger page you will be see another menu of options:
            Welcome to the Ledger-Screen!! 
            Type (A) to display all entries: 
            Type (D) to see deposit entries: 
            Type (P) to see payment entries: 
            Type (R) to go to the report page 
            Type (H) to return back to Home Page 

            1. If you type "A" all of the transactions will be showcased.
            2. If you type "D" only the deposited transactions will be showcased.
            3. If you type "P" only the payment transactions will be showcased.
            4. If you type "R" you will be entering the report screen.
                    Welcome to the Report-Screen!! 
                    Type (1) to display Month to date Report: 
                    Type (2) to display Previous month: 
                    Type (3) to display Year to date: 
                    Type (4) to display previous year 
                    Type (5) to Search by vendor
                    Type (6) to do a custom Search
                    Type (0) to go back to the ledger page

                    1. if you type "1", The Month to date report will be displaying.
                    2. if you type "2", the previous month report will be displaying.
                    3. if you type "3", the year to date report will be displaying.
                    4. if you type "4", the previous year report will be displaying. 
                    5. if you type "5", search by vendor
                    6. if you type "6", you can do a custom search
                    7. if you type "0", you will be going back to ledger
                    
            5. If you type "H" you will return back to the home screen.
       
    5. you can keep operating here until you hit "x" to exit.
